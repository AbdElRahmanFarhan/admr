## **Description**

**Chapter 3**: Simulating Robots Using ROS and Gazebo 

Teleoperation of a mobile robot using keyboard teleoperation

_Original source code_ from [Josep Lentin](https://www.linkedin.com/in/lentinjoseph/): [Mastering Ros for Robotics Programming]( https://github.com/qboticslabs/mastering_ros)

### **Installation** 
Download this package in the _src_ folder of your ROS workspace

```git clone https://github.com/jocacace/diff_wheeled_robot_control```

#### **Dependencies** 
This package depends from the ROS _joy_node_. To install it on Ubuntu 16.04

```sudo apt-get install ros-kinetic-joy```
